# DevOps Automation Projects

This repository contains Python automation tools built as part of my DevOps and Cloud learning journey.

## 🚀 Projects

### 1. Excel Automation
- Cleans and processes Excel files

### 2. Log Analyzer
- Parses log files
- Counts errors, warnings, and info messages
- Generates reports

### 3. Multi-Log Analyzer (CLI)
- Processes multiple log files
- Command-line interface
- Outputs summary report

### 4. Real-Time Log Watcher
- Monitors logs continuously
- Detects new errors and warnings

### 5. Docker Integration
- Containerized Python scripts
- Runs in isolated environment
